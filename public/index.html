What is code-space.me?

Are you a developer looking for an awesome name for your personal website or projects? code-space.me gives you a free yourname.code-space.me subdomain that points to your static site via a CNAME DNS record.

The entire registry lives in a public GitHub repository. You claim your subdomain by submitting a pull request. Works with GitHub Pages, Vercel, Netlify, and other static hosts.

me.code-space.me bd.code-space.me yourname.code-space.me

How to claim your subdomain

Four steps, all done through GitHub.

01

Star and fork

Star the repository, then fork it to your own GitHub account.

02

Set up your site

Deploy your site to GitHub Pages, Netlify, Vercel, or any static host. Note the CNAME value (e.g. yourusername.github.io).

03

Add your subdomain

Create a yourname.json file in the domains/ folder. Edit the JSON to include your username and target CNAME.

04

Submit a pull request

Open a PR against the main repo. Once merged, your subdomain will go live within minutes.

The subdomain entry

Add a JSON object like this to yourname.json.

yourname.json JSON 
{
  "owner": {
    "username": "s8rr",
    "email": "hello@example.com"
  },
  "records": {
    "CNAME": "s8rr.github.io"
  }
}

Replace yourname with your desired subdomain and update the owner details. Point the CNAME value to your hosted site URL.
